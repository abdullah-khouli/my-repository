# To Do App

A new useful flutter application to make your days productive by scheduling your tasks.

## Features and Technologies :

Null Safety                                                                                                                                          
Getx State Management                                                                                                                                                   
Localization                                                                                   
Adaptiveness                                                                          
Responsivity                                                                                               
Light and Dark Theme                                                                                   
Filter By Date                                                                                            
Sqflite Local Storage                                                                                       
MVC Architecture Pattern                                                                                                                  
Local Notifications                                                                                                                           
Splash Screen
